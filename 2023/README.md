# AOC2023
Advent Of Code 2023

This project gathers the solutions of the puzzles from [Advent Of Code 2023](https://adventofcode.com/)

## How to run a solution
It is advised to install and use **Volta** in order to help using node/npm versions according to project's definitions
>[check Volta install instruction](https://docs.volta.sh/guide/getting-started)

Next you must install the project dependencies:
>`npm install`

Then you just need to run the script of the desired day, e.g.:
>`npm run day_03-2`

The solution will be printed in the console
